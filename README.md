### 🩺 Diabetes Prediction — ML Project
<div align="center">
  <img src="https://img.shields.io/badge/scikit--learn-F7931E?style=for-the-badge&logo=scikit-learn&logoColor=white" />
  <img src="https://img.shields.io/badge/Pandas-150458?style=for-the-badge&logo=pandas&logoColor=white" />
  <img src="https://img.shields.io/badge/NumPy-013243?style=for-the-badge&logo=numpy&logoColor=white" />
</div>

A comprehensive **machine learning project** for predicting diabetes risk using the Pima Indians Diabetes Database. Implements multiple classification models with professional-grade practices.

🔹 **Tech Stack:** Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn  
🔹 **Status:** ✅ Completed  
🔹 **Best Model:** KNN Optimized (F1-Score: 0.6250, ROC-AUC: 0.7435)  
🔹 **Key Features:**
- Multiple ML models comparison (KNN, Random Forest, Decision Tree)
- Hyperparameter optimization with GridSearchCV
- Production-ready Python package structure
- Comprehensive EDA and visualization
